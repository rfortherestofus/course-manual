3.3.1 Slides Template

Slides for R for the Rest of Us courses are made using the xaringan package.

There is a custom template to use to make slides in the R for the Rest of Us style. You can access this by installing the rruthemes package using the following code:

devtools::install_github("rfortherestofus/rruthemes")
